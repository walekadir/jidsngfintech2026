JID's DECODE: Nigerian Fintech Landscape 2026

An interactive intelligence report analyzing the 430+ companies re-wiring money in Africa.

 Features
- Unicorn Tracker: Live stats on Nigeria's billion-dollar fintechs.
- Company Directory: Deep dives into Flutterwave, Moniepoint, OPay, and more.
- Consolidation Map: Tracking the surge in M&A deals (67 deals in 2025).
- Interactive Charts: Visualizing valuation, transaction volume, and funding history.

Tech Stack
- Frontend: HTML5, CSS3 (Custom Properties, Grid, Flexbox).
- Animations: JavaScript Intersection Observer API for scroll reveals.
- Design: Editorial style with Glassmorphism and SVG noise textures.

Key Data Points (2026)
- Total Funding: $3.42B raised in 2025.
- Market Share: Nigeria accounts for 28% of all African Fintechs.
- Transaction Scale: ₦800T+ in digital transactions.
